Covid-19 Analysis Using Python

Introduction

The COVID-19 pandemic has had a profound impact on nations and regions across the globe. In India, one of the hardest-hit state is Maharashtra, and this case study explores the dynamics of the pandemic within the state, with a focus on active cases and deaths.

Tools Utilized

The some of the tools we utilized in this project are

NUMPY

NumPy is a Python library for numerical computations, featuring array support and mathematical functions, widely used in data analysis and scientific computing.

PANDAS

Pandas is a Python library for data manipulation, providing powerful data structures like DataFrames. It is a fundamental tool for data analysis and manipulation in various data science and analytics tasks.

MATPLOTLIB

Matplotlib is a Python library for creating static, animated, or interactive visualizations, providing a wide range of customizable 2D and 3D plotting capabilities. It's commonly used for data visualization in scientific and engineering applications.

SEABORN

Seaborn is a Python data visualization library that provides a high-level interface for creating informative and attractive statistical graphics.

Methodology

DATA COLLECTION


This Project Contains two datasets:-

1. Covid-19 India dataset:-That contain States, Confirmed Indian National, Confirmed Foreign National, Cured, Deaths, Confirmed Columns.


2. The second dataset is covid_vaccination_statewise:-That contains states and the vaccination of different age-grouped persons.

DATA PREPROCESSING

Data Analysis
